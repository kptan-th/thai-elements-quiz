# แบบประเมินวิเคราะห์ธาตุเจ้าเรือน - กรุงเทพทิพโอสถ (LINE LIFF App)

เว็บแอปพลิเคชันวิเคราะห์ธาตุเจ้าเรือนตามหลักทฤษฎีการแพทย์แผนไทย พัฒนาขึ้นสำหรับ **บริษัท กรุงเทพทิพโอสถ จำกัด** ทำงานในรูปแบบ **Hybrid Web Application** สามารถใช้งานร่วมกับ **LINE LIFF SDK** เพื่อดึงโปรไฟล์ลูกค้าทักทายในแอป LINE ส่ง Flex Message สรุปผลเข้าห้องแชทส่วนตัว และแชร์ต่อหาเพื่อนได้สะดวก หรือรันผ่านเบราว์เซอร์ปกติภายนอกได้อย่างสมบูรณ์

---

## 🌟 คุณสมบัติเด่น (Key Features)

1. **แบบทดสอบความสมดุลธาตุ 40 ข้อ:** แบ่งเป็น 4 ธาตุหลัก (ดิน, น้ำ, ลม, ไฟ) ธาตุละ 10 ข้อ โดยคำถามอ้างอิงตำราการแพทย์แผนไทย ปรับแต่งถ้อยคำอย่างสุภาพเป็นทางการระดับคลินิกชั้นนำ และออกแบบเพื่อหลีกเลี่ยงอคติคะแนนธาตุไฟเกินจริง (Fire Bias)
2. **กราฟใยแมงมุมแสดงสมดุลธาตุ (SVG Radar Chart):** คำนวณคะแนนรวมสะสมสูงสุด 21.5 คะแนนต่อธาตุ (อิงคะแนนตั้งต้นจากเดือนเกิด 2 คะแนน และคะแนนอาการปัจจุบันสูงสุด 19.5 คะแนน) แสดงผลลัพธ์เป็นโพลีกอนใยแมงมุมที่สวยงามและเข้าใจง่าย
3. **ระบบคำแนะนำเฉพาะบุคคล (Personalized Advice):** แสดงแนวทางดูแลสุขภาพด้านการปฏิบัติตนและอาหารปรับสมดุลตามธาตุเด่นปัจจุบัน รวมถึงระบุคำเตือนกรณีธาตุเจ้าเรือนเกิดดั้งเดิมต่างจากธาตุเด่นปัจจุบัน (ภาวะธาตุกำเริบ/เสียสมดุล)
4. **LINE LIFF Integration (โหมดเปิดใน LINE):**
   - ทักทายต้อนรับด้วยชื่อจริงของผู้ใช้ LINE
   - มีข้อความเตือนเมื่อเปิดนอกห้องแชท และพยายามส่ง Flex Message สรุปผลเข้าห้องแชท LINE ส่วนตัวให้อัตโนมัติเมื่อทำประเมินเสร็จ
   - ปุ่ม **"ชวนเพื่อนมาวิเคราะห์ธาตุเจ้าเรือน"** ปลายทางใช้ระบบ LINE URL Scheme เปิดหน้าต่างแชร์ข้อความหาเพื่อนในแอป LINE ได้ทันทีโดยไม่ติดขัดปัญหาเรื่อง Permission
5. **รองรับการเปิดในเบราว์เซอร์ทั่วไป:** ปิดฟังก์ชันเฉพาะของ LINE และแสดงผลแบบเว็บประเมินทั่วไปอย่างสวยงามลื่นไหล

## 📂 โครงสร้างไฟล์ในโครงการ

* `index.html` - โครงสร้าง HTML5 และการเชื่อมต่อ LINE LIFF SDK
* `style.css` - สไตล์การออกแบบพรีเมียม (Glassmorphism, สไลด์แอนิเมชันคำถาม, ตารางกราฟเรดาร์แบบตอบสนอง)
* `app.js` - ตรรกะจาวาสคริปต์, ชุดคำถาม 40 ข้อ, การคำนวณสูตรธาตุเจ้าเรือน และการควบคุม LIFF SDK
* `logo bangkoktiposod_Send [Converted].png` - โลโก้ทางการของแบรนด์กรุงเทพทิพโอสถ
* `README.md` - คำอธิบายรายละเอียดและคู่มือการใช้งานระบบ

## 🖥️ วิธีการรันเซิร์ฟเวอร์จำลองเพื่อทดสอบบนเครื่องคอมพิวเตอร์ (Local Server)

หากต้องการเปิดใช้งานบนคอมพิวเตอร์เพื่อทดสอบระบบ ให้เปิดโปรแกรม **Terminal (macOS)** หรือ **Command Prompt (Windows)** เข้าไปที่โฟลเดอร์ของโครงการ และรันคำสั่ง:

```bash
# สำหรับ macOS/Linux/Windows (ใช้ Python 3)
python3 -m http.server 8000
```
จากนั้นเปิดเบราว์เซอร์แล้วเข้าไปที่ลิงก์: **[http://localhost:8000](http://localhost:8000)**

## 🚀 การติดตั้งและตั้งค่าใช้งานร่วมกับ LINE LIFF

1. สมัครเข้าใช้งาน [LINE Developers Console](https://developers.line.biz/)
2. สร้าง **Provider** และสร้าง **LINE Login Channel**
3. ไปที่แท็บ **LIFF** แล้วกด **Add LIFF app**
4. นำลิงก์หน้าเว็บที่คุณ Deploy บน GitHub Pages (เช่น `https://your-username.github.io/repository-name/`) ไปกรอกใส่ในช่อง **Endpoint URL** ของ LIFF
5. คัดลอกรหัส **LIFF ID** ที่ระบบ LINE ออกให้
6. เปิดไฟล์ `app.js` ในบรรทัดที่ 3 และกรอกรหัส LIFF ID ของคุณลงไปแทนที่รหัสทดสอบ:
   ```javascript
   const LIFF_ID = "ใส่รหัส-LIFF-ID-ของคุณตรงนี้";
   ```
7. อัปเดตไฟล์ขึ้น GitHub Pages อีกครั้ง ระบบจะพร้อมเชื่อมต่อกับ LINE ทันที
